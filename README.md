Un projet temporaire en mémoire a été créé afin de garantir un environnement de test isolé et non persistant.

L’onglet Proxy a été sélectionné, puis la section Intercept a été vérifiée.

L’état « Intercept is off » a été volontairement maintenu afin d’éviter tout blocage du trafic durant la phase de configuration.
